# 🎓 Alumni Network Website

A modern, responsive alumni platform built for **Thamarai International School** to showcase alumni achievements, events, and institutional updates.

---

## 🚀 Features

### 🏠 Home Page
- Hero section with image slider
- Glassmorphism cards for quick navigation
- Interactive India map showing alumni distribution
- Leadership message carousel

### 👥 Alumni Directory
- Search alumni by name/company
- Filter by batch and college
- Clean card-based UI with icons

### 🎉 Events & Reunions
- Separate sections for upcoming and past events
- Responsive grid layout

### 📢 Announcements
- Institutional updates with date and description
- Clean list view

### 🖼️ Gallery
- Image grid layout
- Hover zoom effects

### 📍 Alumni Map
- Interactive India map using Leaflet
- Color-coded markers:
  - 🔴 IIT
  - 🔵 NIT
  - 🟢 AIIMS
  - 🟠 Other institutions
- Popup with alumni details

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Icons:** lucide-react
- **Map:** react-leaflet
- **Fonts:** Inter (Google Fonts)
